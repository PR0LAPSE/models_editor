все изменения в названиях и именах файлов сразу вносятся в json и переименовывают файлы в локальном репозитории!

  • Поиск вверху слева работает по всем вкладкам и показывает сразу модель в категории
  • Клик по названию модели → изменить название
  • Двойной клик → посмотреть список файлов модели и изменить имена файлов
  • Кнопка доступна только для моделей с количеством файлов > 1, и она удаляет запись о файле из json, но файл в репозитории останется и переименуется с суффиксом "_deleted"
  • Окошко со списком файлов перетаскивается куда удобно
  • Кнопка вверху справа → коммит&пуш в репозиторий
